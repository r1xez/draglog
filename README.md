# DragLog — Платформа для трекінгу швидкості автомобілів

**DragLog** — це  платформа для водіїв , яка дозволяє відстежувати результати заїздів вашого автомобіля та порівнювати результати в глобальному лідерборді.

##  Основні можливості

-   **Глобальний лідерборд**: Перегляд найкращих результатів прискорень (0-100 км/год, 100-200 км/год, 400м) з можливістю фільтрації за маркою, типом палива та приводу.
-   **Мій Гараж**: Можливість додавати власні авто, вказувати специфікації (потужність, об'єм двигуна) та зберігати  модифікації.
-   **Авторизація**: Система реєстрації для пілотів для керування власними даними.

##  Технології

-   **Backend**: Django 6.0.6
-   **Database**: PostgreSQL
-   **Frontend**: HTML5, CSS3 
-   **Containerization**: Docker & Docker Compose

##  Як запустити проект локально

1. **Клонуйте репозиторій**:
   ```bash
   git clone https://github.com/r1xez/draglog.git
   
   ```

2. **Створіть файл `.env`**:
   У корені проекту створіть файл `.env` та додайте туди змінні, що описані в `docker-compose.yml`. 
   

3. **Налаштуйте віртуальне середовище**:
   ```bash
   python -m venv .venv
   source .venv/Scripts/activate  
   pip install -r requirements.txt
   ```

3. **Запуск через Docker**:
   Проект вже містить налаштований `docker-compose.yml` для бази даних PostgreSQL. Не потрібно нічого створювати вручну, достатньо запустити:
   ```bash
   docker-compose up -d
   ```
   Це автоматично підніме контейнер з базою даних, використовуючи налаштування з вашого `.env` файлу.

4. **Виконайте міграції**:
   Після того, як база даних у Docker запуститься:
   ```bash
   python manage.py migrate
   ```

5. **Запустіть сервер**:
   ```bash
   python manage.py runserver
   ```

##  Структура проекту

-   `accounts/` — керування користувачами та профілями.
-   `garage/` — логіка гаража, додавання та редагування авто.
-   `leaderboard/` — відображення та фільтрація результатів заїздів.
-   `static/` — CSS стилі та статичні файли.
-   `templates/` — HTML шаблони сторінок.
